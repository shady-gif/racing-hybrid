# Shadyy

A browser-based single-player arcade racing prototype built with Three.js,
Vite, and Ammo.js.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:5173/game.html`.

## Production build

```bash
npm run build
```

Vercel can deploy the repository with the default Vite settings. See
[ATTRIBUTION.md](./ATTRIBUTION.md) for upstream code and model credits.
